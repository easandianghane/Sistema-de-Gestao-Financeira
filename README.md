# Sistema-de-Gestao-Financeira
Funcionalidades do sistema
•	Criar despesas e categoria de importância (essencial, importante, opcional).
•	As despesas não podem ultrapassar o orçamento. 
•	As Despesas essenciais devem ser totalmente cobertas antes que os importantes ou opcionais sejam consideradas e as opcionais só serão cobertas depois que as outras despesas forem cobertas.
•	Não pode haver salto de meses. Compensa sempre o mês mais baixo.
•	Quando se efetua um pagamento, antes é efetuado o pagamento automático das despesas de importância acima e ou pagamentos do mês anterior.
•	O orçamento definido mensalmente, mesmo que o saldo restante seja elevado, as despesas mensais devem estar dentro do orçamento definido para o referido mês;
•	O saldo remanescente passa para o mês seguinte.
